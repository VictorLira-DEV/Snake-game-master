## how to download this Project
```bash
    #clone the repository
    $ git clone https://github.com/Victor-Lira-DEV/Snake-game-master
    #enter the directory
    $ cd Snake-game-master
    #install the dependencies
    $ yarn install
    #start the project
    $ yarn start
```
